# artshorttext
 
Dieses Modul zeigt den Artikel Kurztext in der Gallerie Ansicht und kann als Beispiel / Vorlage verwendet werden.
Du kannst dieses Modul frei kopieren und modifizieren.

Was ich alles in diesem Modul mache:
